## Python Environment

This project uses a `uv`-managed virtual environment.

- **Location:** `.venv/`
- **Activate:** `source .venv/bin/activate`
- **Install packages:** `uv pip install <package>`
- **Installed packages:** `google-auth`, `google-auth-oauthlib`, `google-api-python-client`
